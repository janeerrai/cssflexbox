Basic Introduction

1. Flex uses one dimenaional layout that is either row or column 2. Flex item can start from the start edge of the main axis 3. Flex item will overflow if the item is more than the screen space it fits. 4. Items will stretch to fill across the main axis in a row 5. flex item do not stretch on the main axis but can shrink.

Flex Direction

There is mainly four type of flex Direction: 1. row, 2. column, 3. row-reverse 4. column reverse
Flex Basis
Flex basis set the width of the flex item.
